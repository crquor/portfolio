C Utilities Documentation

Integer Operations

Count Digits

This function returns the number of digits in the given integer.

Inputs: int num

Usage: countDigits(num);

Factorial

This function returns the factorial of a number. The factorial of a number is the product of all positive integers less than or equal to the number.

Inputs: int num

Usage: factorial(num);

Swap

This function swaps the values of two variables.

Inputs: int *a, int *b

Usage: swap(&a, &b);

Number Type Checks

Is Armstrong

This function checks if a number is an Armstrong number. An Armstrong number is a number that is equal to the sum of its own digits each raised to the power of the number of digits.

Inputs: int num

Usage: isArmstrong(num);

Is Even

This function checks if a number is even. An even number is divisible by 2.

Inputs: int num

Usage: isEven(num);

Is Odd

This function checks if a number is odd. An odd number is not divisible by 2.

Inputs: int num

Usage: isOdd(num);

Is Palindrome

This function checks if a number is a palindrome. A palindrome is a number that reads the same backward as forward.

Inputs: int num

Usage: isPalindrome(num);

Is Prime

This function checks if a number is prime. A prime number is a number greater than 1 that is divisible only by 1 and itself.

Inputs: int num

Usage: isPrime(num);

Array Operations - Integer

Reverse Array

This function reverses the elements of a single-dimension numeric array horizontally.

Inputs: int arr[], int size

Usage: reverseArray(arr, size);

Rotate Array

This function rotates the elements of a single-dimension numeric array by a specified key.

Inputs: int arr[], int size, int key

Usage: rotateArray(arr, size, key);

Max in Array

This function returns the maximum value in a given array.

Inputs: int arr[], int size

Usage: maxInArray(arr, size);

Min in Array

This function returns the minimum value in a given array.

Inputs: int arr[], int size

Usage: minInArray(arr, size);
