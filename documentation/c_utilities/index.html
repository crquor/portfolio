C Utilities Documentation

Swap

This function can be used to swap the values of two variables.

Inputs: int *a, int *b

Usage: swap(&a, &b);

Is Prime

This function returns true if the number is a prime number, and it returns false if the given number is not a prime number

Inputs: int num

Usage: isPrime(num);

Is Armstrong

This function returns true if the number is a armstrong number, and it returns false if the given number is not a armstrong number

Inputs: int num

Usage: isArmstrong(num);

Is Palindrome

This function returns true if the number is a palindrome number, and it returns false if the given number is not a palindrome number

Inputs: int num

Usage: isPalindrome(num);

Reverse Array

This function reverses the elements of a single-dimension numeric array horizontally.

Inputs: int arr[], int size

Usage: reverseArray(arr, size);
